# **App Name**: SubLime Marketplace

## Core Features:

- Subscription Listing: Display available digital subscriptions (Netflix, Spotify, etc.) using a card-based layout.
- Category Browsing: Allow users to browse subscriptions by category (e.g., Entertainment, Music, Productivity).
- Product Details: Show detailed information about each subscription, including price, features, and instructions.
- Order Placement: Guide users through the process of ordering a subscription with clear instructions and support contact.
- Payment Processing: Enable secure payment processing for subscription purchases. Needs integration with Stripe.

## Style Guidelines:

- Primary color: Deep Green (#2E8B57) for a premium feel reminiscent of Apple's design.
- Background color: Charcoal (#121212) to provide a dark mode dominant theme.
- Accent color: Soft Gray (#A9A9A9) to provide subtle highlights and strokes.
- Font: 'Inter', a sans-serif font with a modern, neutral look; for both headlines and body text. Note: currently only Google Fonts are supported.
- Card-based layout with frosted glass/glassmorphism effect, heavy background blur, rounded corners (20-28px), and subtle depth shadows.
- Subtle parallax on scroll, fade + slide transitions, and card hover scale (1.03) for an iOS-like smooth experience.
- Use simple, modern icons for categories and product features, in line with Apple's minimalist design language.
